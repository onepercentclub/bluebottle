📧 Email Preview Gallery - Quick Access
========================================

✅ The preview gallery is now accessible!

🌐 URL (Multi-Tenant):
   http://onepercent.localhost:8000/static/assets/email_previews/index_all.html
   
   Replace "onepercent" with your tenant name if different.

🚀 Start Server:
   python manage.py runserver --settings=bluebottle.settings.local

📝 Generate Previews:
   python manage.py preview_all_messages --all --all-languages

💡 Alternative (Direct File):
   xdg-open bluebottle/notifications/static/email_previews/index_all.html
